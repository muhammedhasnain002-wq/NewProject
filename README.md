# CRM-API

A Customer Relationship Management (CRM) API project built with ASP.NET Core Minimal APIs.

## Project Description
This project provides a robust and efficient API for managing customer data, orders, opportunities, and products within a CRM system. It leverages modern ASP.NET Core features and follows clean architecture principles.

## Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/muhammedhasnain002-wq/CRM-API.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd CRM-API
    ```
3.  **Restore dependencies:**
    ```bash
    dotnet restore
    ```
4.  **Build the project:**
    ```bash
    dotnet build
    ```

## Usage

1.  **Run the application:**
    ```bash
    dotnet run --project src/CRM.Api/CSAPI.csproj
    ```
2.  **Access Swagger UI:**
    Once the application is running, open your web browser and navigate to `http://localhost:5276/swagger` to explore the API endpoints and interact with them.

## Endpoints
The API exposes endpoints for managing:
*   Customers
*   Orders
*   Opportunities
*   Products

Refer to the Swagger UI for detailed information on each endpoint, including request/response schemas and available operations.
